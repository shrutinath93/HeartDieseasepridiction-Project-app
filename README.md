# Heart Disease Prediction System ❤️
This is a Machine Learning web application built using Streamlit that predicts the likelihood of heart disease based on various medical parameters.

🚀 Live Demo
You can access the live web app here: [https://heartdieseasepridiction-project-app-khfaswlkxmag7hbcqghzd7.streamlit.app/]

📌 Project Overview
The goal of this project is to provide a quick and user-friendly way to assess heart health. It uses a K-Nearest Neighbors (KNN) classification model trained on clinical data to identify potential risks.

🛠️ Features
User-friendly Interface: Easy sliders and dropdowns for data input.

Real-time Prediction: Instant results showing whether the user is at risk or not.

Scalable Architecture: Built with Python, Scikit-learn, and Streamlit for easy deployment.

📂 Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-learn, Joblib

Framework: Streamlit

Deployment: Streamlit Community Cloud

⚙️ How it Works
Input Data: The user enters clinical details like Age, Sex, Chest Pain Type, Cholesterol levels, etc.

Preprocessing: The input is processed using a pre-trained StandardScaler to match the training data format.

Prediction: The KNN Model analyzes the features and outputs a prediction.

Result: A success message or warning is displayed on the screen.
